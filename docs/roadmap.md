# SwiftChat Roadmap & Strategy

The development of SwiftChat follows a modular, section-by-section approach. Each major functional area is first prototyped and then thoroughly refined and polished.

## Phase Overview

| Phase | Title | Focus | Primary Goal |
|---|---|---|---|
| **1** | **Secure Foundation** | Networking & Security | Two devices can talk securely. (COMPLETE) |
| **2** | **Gossip Engine** | Mesh & Database | Messages survive disconnection. (COMPLETE) |
| **3** | **Visual Identity** | Branding & Design | Neubrutalist Design System. (COMPLETE) |
| **4** | **Explore & Radar (Detailed)** | Discovery UX | Immersive spatial discovery UI. |
| **5** | **Chat & Social (Detailed)** | Messaging UX | High-end chat interface & Groups. |
| **6** | **Profile & Settings (Detailed)**| Identity & Privacy | Comprehensive user controls. |
| **7** | **Polish & Optimization** | Performance | Battery, Storage, and Tactile Feel. |

## Milestones

### Milestone 3: The "Brand Soul" ✅
- [x] Theme Engine with Light/Dark Neubrutalist support.
- [x] Custom "Physical" component library (Buttons, Cards).
- [x] Redesigned shell with Chat-Centric horizontal swiping.

### Milestone 4: The "Radar Experience" (Phase 4)
- [ ] Immersive Radar view with interactive proximity bubbles.
- [ ] Profile mini-cards and playful "Hi" animations.
- [ ] "Everyone / Contacts Only" privacy implementation.

### Milestone 5: The "Messaging Hub" (Phase 5)
- [ ] Redesigned Chat UI with rich media support.
- [ ] Public and Private group chats.
- [ ] Topic-based matching and auto-discovery.

### Milestone 6: The "Identity Center" (Phase 6)
- [ ] Advanced Profile editing and topic management.
- [ ] Security settings: Peer blocking and data wiping.
- [ ] Offline Distribution: App sharing via Wi-Fi.

### Milestone 7: "Production Ready" (Phase 7)
- [ ] Battery-intelligent discovery cycles.
- [ ] Global storage management (Cache limits).
- [ ] Final tactile polish: Background noise & depth shadows.
