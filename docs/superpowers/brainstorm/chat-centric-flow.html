Interaction Logic: The "Discovery Handle"

Opportunistic Radar

The Radar isn't a separate tab; it's a "curtain" that sits behind or above the chat list. It feels like a tool you pull out when you need it.

Auto-Radar

If the user has 0 active chats, the app defaults to the Radar view with a "Welcome" state to prompt discovery.
