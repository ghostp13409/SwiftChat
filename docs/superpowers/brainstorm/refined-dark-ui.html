UI Design Principles

1. High Physicality

Every interactive element has a hard shadow and thick border, making it feel like a real object you can press.

2. State-Driven Color

Primary colors aren't just for branding; they communicate status (Green=Active, Purple=Mesh, Red=Error).

3. Extreme Clarity

Bold, uppercase typography for primary actions to reduce reading time and cognitive load.

4. Bouncy Feedback

Interface elements use elastic animations (Springs) instead of linear fades.
