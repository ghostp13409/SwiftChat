# Phase 6: Detailed Profile & Settings

**Goal:** Provide comprehensive user control over identity and privacy within the mesh.

## Requirements
- Full-featured Profile editor.
- Granular Privacy settings.
- Offline distribution tools.

## Technical Tasks

### 6.1 The Identity Center
- [ ] Implement avatar selection/upload.
- [ ] Topic management (Add/Remove tags).
- [ ] Bio editor with real-time preview.

### 6.2 Privacy & Security
- [ ] Peer Blocking (Block and unblock specific IDs).
- [ ] Visibility management (Persistent state for "Everyone" vs "Contacts Only").
- [ ] Data Management: Detailed storage breakdown and "Self-Destruct" (Wipe all data).

### 6.3 Distribution
- [ ] "Share the App" feature: Build-in Wi-Fi Direct file sharing of the app's own APK/Installer.

## Success Criteria
1. Users can manage their identity without internet.
2. Blocking a peer immediately hides them from Radar and stops syncing their messages.
3. The app can be shared with a nearby friend completely offline.
